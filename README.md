# Auctionator Mini Search (Retail)

Auctionator Mini Search is a lightweight companion addon for Auctionator that lets
you quickly search indexed auction prices in a compact in-game window.

## Dependency

This addon requires Auctionator to be installed and enabled:
https://www.curseforge.com/wow/addons/auctionator

## Important Notes

- Retail version only.
- Auctionator must have available price data for results to appear.
- Some item names become available only after they are cached by the WoW client.
